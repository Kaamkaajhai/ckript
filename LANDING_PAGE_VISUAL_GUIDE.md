## ⚙️ Animation Details

### Entrance Animations (Page Load)
```jsx
Initial: opacity: 0, y: 20
Animate: opacity: 1, y: 0
Duration: 0.8s, ease: "easeOut"
Stagger: 0.2s between items
```

### Scroll Animations (whileInView)
```jsx
Initial: opacity: 0, y: 20
Animate: opacity: 1, y: 0
Duration: 0.5-0.6s
Trigger: When element enters viewport
```

### Hover Effects
```jsx
Button: scale(1.05), shadow increase
Card: border color change, background shift
Icon: scale(1.1), color change
```

### Transitions
```jsx
All: 0.3s duration
Timing: ease-in-out
GPU Accelerated: Uses `transform` and `opacity`
```
